hike.io
=============

A simple sinatra app for finding [beautiful hikes](http://hike.io).

Setup
-------------

1. Install [ruby](www.ruby-lang.org)
2. Install [node](http://nodejs.org) & [npm](https://npmjs.org/)
3. Install and open [Postgres.app](http://postgresapp.com)
4. `npm install`
5. `bundle install`
6. `rake`
